# Bridge Card — 포트폴리오 / 이력서용 프로젝트 카드

> 원본은 회사 내부 ZOOM 운영 도구. 아래 문구는 포트폴리오용으로 일반화했습니다.

---

## 한 줄 요약

ZOOM 화면 공유 기반 아이스브레이킹 웹 도구. Vue.js 카드 게임 로직·타이머·2단계 대화 모드, 리뷰 API 연동 및 온보딩 콘텐츠 개발.

---

## 짧은 버전 (이력서 3~4줄)

**온라인 브릿지 카드** | Vue 3 · TypeScript · Vite  
- 코로나 비대면 환경에서 ZOOM 화면 공유용 아이스브레이킹·대화 진행 웹 도구 기획 및 프론트엔드 개발  
- 카드 선택·제거·라이트/딥 모드·타이머 등 게임 로직 구현으로 진행자 주도 세션 운영 지원  
- 리뷰 작성/게시판 REST API 연동, 유튜브 가이드 영상 연동으로 사용자 온보딩 체계 구축  
- 포트폴리오 버전: Vue 3 + TypeScript + Tailwind로 리팩토링, 서버 기능 Mock 처리

### 담당 업무

- ZOOM 화면 공유에 최적화된 카드 UI (선택 → 풀스크린 → 제거 흐름)
- 라이트/딥 모드 2단계 대화 진행 로직
- 타이머, 카드 셔플(Fisher-Yates), 다국어(한/영) 지원
- 리뷰 작성·게시판 페이지 및 REST API 연동 (이미지 업로드)
- 유튜브 사용 가이드 영상 연동

### 기술 하이라이트

1. **게임 상태 관리** — Composition API composables로 덱·타이머·모드 분리
2. **애니메이션** — jQuery 제거, Vue 반응형 + CSS transition 카드 셔플
3. **접근성** — ESC 모달 닫기, ARIA 라벨, reduced-motion 지원
4. **포트폴리오 리팩토링** — Vue 2 → Vue 3 + TypeScript + Tailwind 마이그레이션

## 면접 설명용

### 배경
회사 특성상 많은 신규 만남이 있었고, 코로나로 ZOOM 비대면 세션이 늘면서 화면 공유로 진행할 아이스브레이킹 도구가 필요했습니다.

### 해결
브릿지 질문 카드를 웹으로 구현해, 진행자가 카드를 하나씩 선택하고 선택된 카드는 제거되며 다음 단계로 더 깊은 대화를 이어가도록 설계했습니다.

### 포트폴리오 공개
원본은 회사 서비스이므로 API·리뷰 기능은 Mock 처리한 sanitized 버전을 별도 레포로 공개했습니다.
